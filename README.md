# ShamsharWiredBrainCoffee

EmployeeManager - Employee Management System
A Blazor .NET 8 web application for managing employee records with full CRUD (Create, Read, Update, Delete) functionality. This application features a clean, modern interface for handling employee data with a local database backend.

Features
Complete CRUD Operations: Create, view, edit, and delete employee records

Modern UI: Built with Blazor components for a responsive user experience

Local Database: Uses a local database for data persistence (SQL Server/SQLite)

Employee Management: Track employee details, departments, and contact information

.NET 8: Leverages the latest .NET 8 features and performance improvements




Tech Stack
Frontend: Blazor (Interactive Server/Render Mode)

Backend: ASP.NET Core .NET 8

Database: Entity Framework Core with LocalDB/SQLite

Architecture: Clean separation of concerns with data models and business logic
